Driver's Test Program
Luke Swindells (9855467)
Expected Grade: Credit
To run program open dist folder and open DriversTest.jar
Program opens with terms and allows user to select background colour.
Pressing start runs the test for 5 minutes, the user must click
one of the three cars when a hazard appears in front of them. Click a 
car at the wrong time and it will be marked.
After the test a table will show the results and times of each click or
missed click. The user can then press a button to save the file in .txt
format.   
Colour Selector code based on code from Lab.
